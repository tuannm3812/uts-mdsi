# 94691 Deep Learning - Glossary

| Term | Evidence Count | Working Definition |
|---|---:|---|
| pytorch | 1074 | A Python deep learning framework used to define tensors, neural network modules, losses, optimisers, and training loops. |
| computer vision | 846 | Methods for representing, analysing, and modelling images or visual data. |
| neural network | 330 | A model made of layers of learned parameters that transform inputs into predictions through nonlinear functions. |
| activation function | 305 | A nonlinear function such as ReLU, sigmoid, tanh, or softmax that lets neural networks model non-linear relationships. |
| convolutional neural network | 294 | A neural architecture that uses convolutional filters to learn spatial features from image-like inputs. |
| optimisation | 245 | The process of updating model parameters, often with SGD or Adam, to reduce the loss function. |
| transfer learning | 172 | Reusing a pretrained model or representation for a new task, often by freezing or fine-tuning selected layers. |
| image captioning | 160 | A multimodal task that generates text descriptions for images, often using an image encoder and text decoder. |
| evaluation | 155 | The process of judging model performance with metrics, validation data, error analysis, and task-specific criteria. |
| loss function | 114 | The objective being minimised during training, such as cross-entropy for classification or mean squared error for regression. |
| object detection | 90 | A computer vision task that predicts both object classes and object locations, usually via bounding boxes. |
| inception | 76 | A CNN architecture pattern that combines multiple convolutional filter sizes to capture features at different scales. |
| backpropagation | 66 | The algorithm used to compute gradients of a loss with respect to model parameters so the optimiser can update weights. |
| regularisation | 64 | Techniques that reduce overfitting, including dropout, weight decay, data augmentation, and early stopping. |
| resnet | 64 | A CNN architecture family that uses residual connections to make deeper networks easier to train. |
