# LLM Wiki Workflow

## Ingest

1. Read lecture slides, notebooks, readings, and assessment briefs from the Google Drive subject folder.
2. Add concise notes to the relevant subject page.
3. Extract reusable ideas into `03-shared-concepts/`.
4. Add source file names and dates so claims remain traceable.

## Study

1. Ask the LLM for beginner, technical, and applied explanations.
2. Generate quizzes from a subject page or concept page.
3. Ask for cross-subject links before assignments and exams.
4. Move durable insights back into Markdown.

## Assessments

1. Create one assessment note per task in `04-assessments/`.
2. Paste the official task and rubric into the note.
3. Ask the LLM to convert the rubric into a checklist.
4. Use the LLM to critique drafts, then verify all citations, claims, code, and calculations yourself.

## Weekly Maintenance

- Update active subject pages.
- Add new concepts discovered in lectures.
- Fill the assessment calendar.
- Add prompts that worked well to `06-prompt-library/master-prompts.md`.
