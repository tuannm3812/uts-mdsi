# Master Prompts

## Lecture To Wiki Notes

Summarise this lecture into key concepts, formulas, examples, assumptions, common mistakes, and links to existing wiki pages. Keep source references explicit.

## Assignment Rubric Reviewer

Act as a strict marker. Compare my draft against the official rubric. Return missing requirements, weak evidence, unclear reasoning, and concrete revisions.

## Exam Drill

Create 20 questions from this subject page: 10 conceptual, 5 applied, and 5 calculation or code-based. Provide answers separately.

## Cross-Subject Synthesis

Connect this topic to statistics, machine learning, data engineering, visualisation, deep learning, NLP, generative AI, and reinforcement learning where relevant.

## Code Explainer

Explain this notebook code cell by cell. Identify the data flow, assumptions, bugs, and what I should be able to explain in an assessment.
