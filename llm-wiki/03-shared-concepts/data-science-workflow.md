# Data Science Workflow

## Definition

The practical loop of framing a problem, obtaining data, cleaning it, exploring it, modelling it, evaluating results, and communicating decisions.

## Checklist

- Problem framed
- Data sources understood
- Cleaning decisions recorded
- Model or analysis justified
- Evaluation matched to the real task
- Findings communicated clearly
