# Data Visualisation

## Definition

The use of visual encodings to communicate patterns, comparisons, uncertainty, and analytical arguments.

## Review Questions

- What question does the chart answer?
- Is the encoding appropriate for the data type?
- Does the visual support or exaggerate the claim?
- What context is missing?
