# Prompt Library

## Purpose

Store prompts that reliably help with study, assessment review, coding, and synthesis.

## Main File

- [Master Prompts](../06-prompt-library/master-prompts.md)

## Prompt Quality Checklist

- Defines the role
- Provides task context
- Includes source material
- Specifies output format
- Asks for uncertainty and assumptions
- Requires verification steps
